## Render Deployment

### 1. Account Setup
- [ ] Render account created at render.com
- [ ] GitHub connected to Render
- [ ] Payment method added (if using paid plan)

### 2. Web Service Configuration

**Basic Settings:**
- [ ] Name: `schoolsync-pro` (or your choice)
- [ ] Environment: `Python 3`
- [ ] Region: Choose closest to your school
- [ ] Branch: `main`

**Build & Deploy:**
- [ ] Build Command: `pip install -r requirements.txt`
- [ ] Start Command: `gunicorn --worker-class eventlet -w 1 app:app`

**Instance Type:**
- [ ] Free (for testing/small schools)
- [ ] Starter ($7/month - recommended for production)
- [ ] Standard ($25/month - for larger schools)

### 3. Environment Variables

**Required Variables:**
```
SECRET_KEY = [Generate 64-character random string]
DEBUG = False
PORT = 10000
```

**Optional Variables (for production):**
```
DATABASE_URL = [PostgreSQL URL if upgrading from SQLite]
UPLOAD_FOLDER = uploads
MAX_CONTENT_LENGTH = 16777216
```

**Generate SECRET_KEY:**
```python
import secrets
print(secrets.token_hex(32))
# Copy output to Render environment variable
```

- [ ] All environment variables added
- [ ] SECRET_KEY is unique and secure
- [ ] DEBUG is set to False

### 4. Deploy
- [ ] Click "Create Web Service"
- [ ] Wait for build to complete (2-5 minutes)
- [ ] Check deploy logs for errors
- [ ] Verify deployment successful
